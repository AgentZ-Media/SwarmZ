## SwarmZ v0.2.7 — voice dictation, custom commands & workspace presets 🎙️

### New

- 🎙️ **Voice dictation** — hold plain `⌘` as push-to-talk (or switch to
  `⌘⇧M` toggle mode), or click the mic in any pane or floating-terminal
  header, and just speak. A live waveform pill shows the recording; on
  release the audio is transcribed via OpenRouter speech-to-text and
  pasted into that terminal — review and hit Enter, or enable
  auto-submit. An optional LLM cleanup pass strips filler words with an
  editable prompt (never translates). Needs an OpenRouter API key
  (Settings → Voice) — stored in the macOS Keychain, all requests run
  natively; without a key the mic UI stays hidden.
- 📋 **Custom commands** — `⌘⇧K` opens an insert picker with your saved
  prompt snippets, global or per project folder. Selecting one pastes it
  into the active pane without submitting (`⌘↵` pastes & runs), so you
  can still edit before sending. Snippets support placeholders:
  `{{folder}}`, `{{cwd}}`, `{{branch}}`, `{{agent}}` and
  `{{input:Label}}` for a value asked right before inserting. Manage
  them in Settings → Commands, or save new ones straight from the
  picker.
- 🧩 **Workspace presets** — every empty workspace now offers preset
  cards with a mini layout thumbnail; one click spawns the whole grid
  (`⌘⇧N` → pick a card = a full new setup in two keystrokes). Ships
  with editable starter grids (Solo, 1×2, 2×2, 1+2); save your own via
  `⌘K` → **Save workspace as preset**. Panes can pin a fixed folder or
  inherit one asked once at load. Manage everything in Settings →
  Presets.

### Changed

- **Session restore is now off by default** — enable it in Settings if
  you want panes and conversations back after a relaunch.
- The quit guard now warns whenever closing would lose something: agents
  still working always count, and idle-but-open panes count too while
  session restore is off (with restore on they come back anyway).
