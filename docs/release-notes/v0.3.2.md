# v0.3.2 — pick your microphone

A small quality-of-life release for voice dictation.

## 🎙️ Microphone picker

- **Settings → Voice → Microphone**: choose which input device dictation
  records from — built-in mic, headset, audio interface. The default keeps
  following the input selected in macOS Sound settings.
- The device list is fetched only when the picker is opened (that's also
  when macOS may ask for mic permission the first time), and your choice is
  remembered across restarts.
- **Safe fallback**: if the saved mic isn't connected (headset left at
  home), recordings automatically fall back to the system default instead
  of failing — the Settings picker shows the saved device as
  "(not connected)" until it's plugged back in.
- Works with both transcription engines, cloud (OpenRouter) and local
  (Parakeet).
