## SwarmZ v0.1.3 — auto-named panes, live Claude status & pane zoom 🏷️

### New

- **Auto-naming** 🏷️ — Claude Code generates a topic title for every session
  (and updates it on `/rename`); SwarmZ captures it from the terminal title
  and names the pane after it — no more guessing which "Agent 3" is doing
  what. Rename a pane yourself (double-click) and the auto-title backs off;
  clear the name to hand naming back to Claude.
- **Live status dot** 🚦 — the dot in each pane header now mirrors what
  Claude is actually doing: amber while it's working, green when idle, blue
  when it waits for your input. Captured straight from Claude Code's
  terminal progress reporting (plus the bell) — no polling, works for every
  pane at once.
- **Per-pane zoom** 🔍 — `⌘+` / `⌘-` change the active pane's font size,
  `⌘0` resets it. Each pane remembers its own zoom.

### Under the hood

- SwarmZ now advertises terminal progress support to programs running in
  its PTYs, which is what makes Claude Code emit its busy/idle state.
- Claude Code's native tab-status protocol (idle / working / waiting) is
  already wired up — the moment Anthropic enables it, panes will pick up
  the full status including "waiting for input" automatically.
