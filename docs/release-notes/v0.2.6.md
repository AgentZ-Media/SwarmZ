## SwarmZ v0.2.6 — session restore fixes 🔁

### Fixed

- Panes that never did anything no longer resume a stray session after a
  restart. Session latching is now gated on real activity: a pane only
  picks up a session file once its claude has actually worked, and
  discovery only matches files born around that first activity — so an
  idle pane can't steal (and later resume) a sibling session from the
  same folder.
- Usage stats and the context donut work again right after a restored
  pane resumes: the latched session file is now parsed in full, so the
  pre-restart turns count instead of starting from zero.
