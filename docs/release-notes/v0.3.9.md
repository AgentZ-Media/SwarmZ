# v0.3.9 — Codex CLI agents

- Added ChatGPT Codex CLI as a first-class agent runtime alongside Claude Code
  and plain shells.
- New Codex profiles, runtime selection in the New Agent dialog, Codex binary
  path override, and runtime-aware session restore via `codex resume`.
- Usage tracking now reads Codex rollout JSONL files from `~/.codex/sessions`,
  including model, tokens, context window, thread names, activity state and
  rate-limit windows.
- The usage dashboard, pane stats, title-bar meters and docs now distinguish
  Claude and Codex runtime data.
- Updated pnpm build-script approval config for pnpm 11 so CI/build installs can
  run `esbuild` and `node-pty` scripts without the deprecated package.json
  setting.
