## SwarmZ v0.2.8 — on-device voice dictation 🔒🎙️

### New

- 🔒 **Local speech-to-text engine** — voice dictation can now run fully
  on-device via [NVIDIA Parakeet TDT 0.6b v3](https://huggingface.co/nvidia/parakeet-tdt-0.6b-v3)
  (int8 ONNX, 25 languages) through a statically linked ONNX Runtime: no
  internet, no API key, no extra installs. Pick the engine in Settings →
  Voice and download the ~670 MB model once (resumable, with live
  progress); it loads lazily on the first dictation (~1.5–2 GB RAM while
  resident) and is freed again when you switch back to Cloud. With the
  local engine your recordings never leave the machine. The optional LLM
  cleanup pass still runs via OpenRouter, so it needs a key even with
  the local engine.
- ⌨️ **Custom commands in the ⌘K palette** — your saved prompt snippets
  now also surface in the command palette as soon as you start typing a
  search (folder commands of the target pane + globals), with the same
  paste / `⌘↵`-submits semantics as the `⌘⇧K` insert picker. Snippets
  with `{{input:Label}}` placeholders detour through the picker's inline
  form to ask for their values first.

### Changed

- The README and in-app settings now distinguish the two dictation
  engines (Cloud via OpenRouter / Local via Parakeet); mic availability
  follows the selected engine.
