# v0.3.1 — quick notes

One new feature: capture ideas without leaving SwarmZ.

## 📝 Quick notes

- **⌘N** (or the note button in the title bar) slides in a checklist drawer
  from the right — type, Enter, captured. The input is focused the moment
  the drawer opens, so a thought is saved in two keystrokes.
- Items **check off** (struck through and dimmed), **edit inline** (click
  the text), **reorder by drag** (grip handle on hover) or turn into plain
  text lines without a checkbox. **Clear done** sweeps everything checked.
- **Global + per project**: one list for everything, plus one list per
  project folder. The drawer opens in the context of the active pane —
  worktree panes count as their main repo, so notes follow the project, not
  the throwaway `.worktrees/` checkout. Scope chips at the top switch
  between Global, the active project and every folder that already has
  notes.
- Everything persists immediately and survives restarts (stored in
  `swarmz.json`, covered by the crash-safe backup from v0.3.0).
