# v0.3.11 — Codex-first native release

- Removed the abandoned web backend path and kept SwarmZ native-only.
- Added a Settings default-app control for new panes, with fresh installs
  defaulting to Codex full-access/YOLO mode.
- Added Codex standard profiles for YOLO/full-access, workspace and read-only
  workflows, including backfill for existing profile lists.
- Fixed a React/Zustand render loop in the worktree title-bar button that could
  leave the dev window blank.
- Added a GPT Realtime 2 orchestration-layer plan covering pane context,
  tool access, safety boundaries and phased delivery.
- Split CI so frontend checks run on Ubuntu and Rust/Tauri tests run on macOS.
