## SwarmZ v0.2.2 — drag & drop files, quit guard & live status counts 📎

### New

- **Drag & drop files onto terminals** 📎 — drag an image (or any file)
  from Finder onto a terminal and its path is typed in, escaped exactly
  like Terminal.app does — Claude Code attaches it. While dragging, every
  terminal shows a drop zone and the one under the cursor lights up;
  works on panes and floating terminals alike. Multiple files drop as
  multiple paths.
- **Quit guard** 🛑 — quitting SwarmZ (⌘Q, menu or the window close
  button) while agents are still working now raises a warning that lists
  the busy agents first, so a stray ⌘Q can't kill a swarm mid-task.
- **Live status counts** 🚦 — the title bar now sums up what the swarm is
  doing in real time: `2 working · 1 idle`, with a pulsing amber dot
  while anything is busy — driven by the same signal as the pane status
  dots, no polling.
- **Clearer active pane** 🔵 — the focused pane gets a stronger ring and
  a tinted header, so it's obvious where your keystrokes go.

### Fixed

- Long model names, token counts and costs no longer overflow or shove
  each other around in the usage drawer and stats popovers — everything
  truncates and aligns cleanly.

### Under the hood

- Web mode is officially retired — SwarmZ is a native macOS app. Docs no
  longer reference `pnpm dev:web` or the Node engine.
