## SwarmZ v0.1.0 — first public release ⚡

Run, tile and monitor a swarm of Claude Code agents — real PTY terminals,
live token & cost tracking, 100% local.

### Highlights

- **Real terminals** — every agent is a PTY-backed login shell; `claude`,
  nvm, aliases and your environment resolve exactly like in iTerm/Terminal.
- **Tiling split-grid** — split panes right (`⌘D`) or down (`⌘⇧D`), drag to
  resize. Terminals never remount, scrollback survives.
- **Live usage** — model, tokens and USD cost per agent in real time, parsed
  from `~/.claude/projects/*.jsonl`.
- **All-time statistics** — every Claude session launched inside SwarmZ is
  persisted across restarts (Session / All-time toggle in the usage drawer).
- **Auto-updates** — checks for new versions in the background; manual check
  via the refresh button in the title bar.
- **Profiles & notifications** — startup presets, and a native notification
  when an agent needs your attention.
